Welcome to LaosLaser

The Open Source Laser project. This Github site hosts the source code and various other document of the project. You can browse the source here: http://github.com/laoslaser There are various repositories here:

  • Software - The CUPS printer driver and associated software
  • Hardware - Hardware design files for the main PCB
  • Firmware - Embedded software for the (MBED) ARM controller on the mainboard
  • Documentation - Various documents and reverse engineering documents
  • AddOns - Extra hard and software components for the Laos laser system (e.g. the I2C Display PCB)

  • The main project page is here: http://laoslaser.org

  • There is a WIKI with documentation and other information here: http://redmine.laoslaser.org
